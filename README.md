# Abbott Power BI & OneLake Integration

## Overview
This project demonstrates a complete data integration and reporting pipeline using **Microsoft Fabric (OneLake)** and **Power BI**.  
It was developed as part of a professional showcase for data engineering and BI reporting skills.

## Architecture
- **Python** for data upload to OneLake (Fabric Lakehouse).
- **Power BI** for modeling, analysis, and visualization.
- **Azure CLI & Fabric SDK** for secure connection and authentication.
